
¿Qué comandos necesita usar Fritz para clonar el repositiorio y ponerse al día con los commits hechos en su ausencia?
    git remote add origin /home/jovyan/repos/local
    git pull origin master
    git log






Solución con git reset:
    1. git reset --hard [SHA-1 de static/screenMSG.txt]
    2. Iterar los siguientes pasos para cada commit que se desea incluir:
        2.1. git checkout [SHA-1 del commit que se quiere incluir en la nueva historia] :/
        2.2. git commit -a -m [mensaje de ese commit]
    3. git push origin master --force 
*Esto cambia la autoría de los commits. Sería bueno reflejar eso en los mensajes nuevos
Los SHA-1 se muestran con git log --oneline -graph --all
Los compañeros tendrán que usar rebase para actualizar su trabajo



Solución con git reset:
    1. git reset --hard [SHA-1 de static/screenMSG.txt]
    2. Iterar los siguientes pasos para cada commit que se desea incluir:
        2.1. git cherry-pick [SHA-1 del commit]
        2.2. git commit -a -m [mensaje de ese commit]
    3. git push origin master --force 

Solución con git rebase -i:
    1. Abrir una terminal desde el Launcher (Ctrl+Shift+L)*
    2. git reset --hard [SHA-1 de static/screenMSG.txt]
    3. git rebase -i
        3.1 añadir una línea que contenga: "drop [SHA-1 del commit no deseado]"
        3.2 añadir lineas que contengan: "pick [SHA-1 de los commits no deseado]" (una línea por commit)
*Esto permite a git rebase -i abrir el editor
